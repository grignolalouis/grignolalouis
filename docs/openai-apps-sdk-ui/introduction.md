# Apps SDK UI

> Design system for building high quality apps in ChatGPT

Apps SDK UI is a design system tailored for ChatGPT Apps SDK, providing styling foundations with Tailwind, CSS variable design tokens, and a library of well-crafted, accessible components. Apps SDK UI includes:

- **Design tokens** - for colors, typography, spacing, sizing, shadows, surfaces, and more.
- **Tailwind 4** - pre-configured with Apps SDK UI's design tokens.
- **Component library** - high-quality components built on top of Radix for consistent accessibility patterns.
- **Utilities** - for dark mode, responsiveness layouts, and more.
